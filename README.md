# Guess Name Game 🎮🔠

## Overview 📝
The **Guess Name Game** is a Windows Forms application that allows multiple clients to connect via TCP protocols. Two players can join a game session, while additional users can watch the game in real-time.

## Features 🌟
- 🎭 **Multiplayer Mode**: Only two players can actively participate in a game session.
- 👀 **Spectator Mode**: Additional clients can join as watchers to observe the game.
- 🔄 **Real-time Communication**: Uses TCP protocol for seamless interaction between clients.
- 🖥️ **Windows Forms UI**: Simple and intuitive graphical interface.

## Installation 🛠️
1. Clone the repository:
   ```sh
   git clone https://github.com/your-repo/guess-name-game.git
   ```
2. Open the solution file in Visual Studio.
3. Build and run the project.

## How to Play 🎲
1. Launch the application.
2. Enter your name and connect to the server.
3. If no active game is running, you can start a new session.
4. Wait for an opponent to join.
5. Spectators can join existing sessions to watch the game unfold.

## Requirements 📋
- .NET Framework 4.7+
- Windows 10/11
